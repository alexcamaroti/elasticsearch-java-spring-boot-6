# Produtosearch	

An application with a higher purpose in carrying out combined research and in some cases incomplete with the intention
to return what the user wants.
Simulating Amazon's research engine, the application uses Elasticsearch and Java Springboot in its development.
For visualization of the data, Kibana is being used.

### Prerequisites

[Elasticsearch v6.4.2](https://www.elastic.co/downloads/elasticsearch)

[Kibana v6.4.2](https://www.elastic.co/downloads/kibana)

[Maven](https://maven.apache.org/download.cgi)
